# Todo App Backend

Projetinho de Todolist simples para aprendizado de backend com Javascript.

## Instalação

```bash
$ npm install
```

## Rodando aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Tecnologias utilizadas
- [Nest](https://nestjs.com/)
- [Typescript](https://www.typescriptlang.org/)
